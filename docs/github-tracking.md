# GitHub Tracking Checklist

Use these items to create GitHub issues and project-board cards in the Classroom repository.

## Issues

- `chore: scaffold React project`
  - Create Vite app structure
  - Add package scripts
  - Verify local app starts
- `feat: configure Supabase backend`
  - Create Supabase project
  - Add environment variables
  - Run database schema SQL
- `feat: implement authentication`
  - Add registration form
  - Add login form
  - Add logout flow
  - Verify session persistence
- `feat: add authenticated habit CRUD`
  - Create habits
  - Read user-specific habits
  - Mark habits complete
  - Delete habits
- `docs: complete assignment documentation`
  - Project overview
  - ERD and design decisions
  - Architecture overview
  - Setup instructions
  - Deployment notes
- `chore: deploy application`
  - Add Netlify site
  - Configure production environment variables
  - Smoke-test the live URL
- `docs: record demo video`
  - Show registration
  - Show login
  - Show create, update, and delete actions
